# snake
thinkphp5做的通用系统改后台

目前完成的功能有:

1. 后台管理员的增删改查
2. 角色的增删改查
3. 权限的分配
4. 数据库的备份与还原

# 使用方法
1. 使用`composer`工具建立项目
    `composer create-project xiaosumay/snake [your_project_name]`
2. 开始操作 `php think keygen`
3. 修改数据库配置参数
4. 生成表 `php think  migrate:run`
5. 生成默认数据 `php think seed:run`
